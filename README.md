# II_Materials

A collection of computational work completed as part of Part II Materials Science at the University of Cambridge.  
This repository consolidates code written for practicals, the alloy design project, and supporting work for the Raspberry Pi and supervision exercises.

---

## Repository Overview

| Folder | Description |
|---------|--------------|
| **Alloy_Design/** | Scripts for alloy composition screening, data analysis, and visualisation associated with the Alloy Design Project. |
| **P2_Plasticity/** | Code and data processing for the Part II Plasticity practical, including stressâ€“strain analysis and curve fitting. |
| **P3_Polymer_Crystallisation/** | Tools for analysing polymer crystallisation kinetics and associated experimental data. |
| **RPi_Project/** | Python code and utilities developed for the Raspberry Pi project, including data logging and peripheral control. |
| **Supervisions/** | Miscellaneous scripts written for supervision work and exploratory analysis. |

## Requirements

All code is written in **Python 3** and relies on standard scientific libraries.

### Core dependencies
- `numpy`
- `pandas`
- `matplotlib`
- `scipy`
- `math` (standard library)
- `os` (standard library)
